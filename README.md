## Descripción

Este proyecto está orientado a apoyar el proceso de gestión de calidad mediante la integración con Jira y Zephyr.  
Incluye ejemplos visuales de funcionalidades como backlog, features, y la plataforma Zephyr para gestión de pruebas.  
Sirve como referencia para testers y desarrolladores que trabajan en automatización y seguimiento de pruebas dentro de Jira.

---

## Cómo usar

- Consultá las imágenes para entender la estructura y funcionalidad.  
- Usá los recursos para configurar tu proyecto de QA en Jira con Zephyr.  
- Actualizá este repositorio con más documentación y ejemplos según avances.
